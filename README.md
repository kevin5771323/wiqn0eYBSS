# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行舌象检测
- 提供6类舌象识别：红舌厚腻、白舌厚腻、黑苔、地图舌、紫苔、红舌黄苔厚腻
- 支持自定义置信度和IOU阈值
- 可选择显示标签和原图
- 支持选择检测模型
- 提供批量检测功能，可直接查看所有检测结果
- 支持检测结果保存
- 包含2000张标注数据集
- 提供完整源码、训练好的模型及训练结果
- 提供项目运行教程

## 适用场景
- 中医舌诊辅助分析
- 医学研究数据采集
- 教学演示工具
- 健康检测应用

## 技术特点
- 基于YOLOv8深度学习框架
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 依赖python3.9、opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供图形用户界面，操作简便直观
- 已训练好模型，配置环境后可直接使用
